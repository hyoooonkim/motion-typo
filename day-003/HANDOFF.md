# Day 003 — 세탁기와 AI가 벌어준 시간은 다 어디로 갔을까?

## 작품 개요
- **텍스트**: "세탁기와 AI가 벌어준 시간은 다 어디로 갔을까?"
- **컨셉**: Matter.js 물리 엔진으로 세탁기 드럼 시뮬레이션
- **인터랙션**: 클릭으로 세탁 시작, 자동 종료

## 핵심 메커니즘
실제 세탁기 사진(`image.png`) 위에 드럼 유리 영역만 물리 시뮬. 드럼 안에서 글자들이 회전하며 섞이다가 점점 사라지고, 마지막에 "시간은 다 어디로 갔을까?"만 남음.

## 드럼 좌표 (중요)
사용자가 Illustrator Transform 패널로 실측한 값:
- `DRUM_CX_RATIO = 2059.72 / 4096`
- `DRUM_CY_RATIO = 1539.08 / 3186`
- `DRUM_R_RATIO = (827.44 / 2) / 3186`

이미지가 바뀌면 이 비율도 다시 측정 필요.

## 시퀀스
1. 초기: 모든 글자가 드럼 안에 떠 있음
2. 클릭: 15초간 회전 + 거품이 점점 차오름
3. 회전 중 글자들이 점점 제거됨
4. 종료: "시간은 다 어디로 갔을까?"만 남음

## 기술 스택
- Matter.js (로컬 CDN 또는 인라인)
- Canvas 렌더링
- 회전 속도 제한: `drumSpeed max 0.02` (넘으면 글자가 벽을 뚫고 나감)

## 에셋
- `image.png` — 실제 세탁기 사진 (블랙 배경)
- `화면 기록 2026-04-20 오후 1.28.07.mov` — 데모 영상

## 디자인 방향
- 벡터형 세탁기가 아닌 실사 이미지
- 거품은 점진적으로 차오르다 끝까지 가득 차는 연출

## 주요 이슈/러닝
- Matter.js 회전 속도 제한 필수 (글자가 드럼 벽 뚫고 나감)
- 드럼 좌표 측정 여러 번 반복 — Illustrator Transform 패널 활용이 정답
- 벡터 스타일 세탁기는 사용자가 싫어함 → 실사 이미지로 전환
