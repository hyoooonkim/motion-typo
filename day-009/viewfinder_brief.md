## 컨셉

동일한 말(horse) 이미지 한 장을 **7가지 렌더링 모드**로 처리해 불규칙한 패널 그리드에 배열.  
패널마다 다른 표현 방식이 교차하며 루핑 — "같은 피사체가 동시에 여러 세계에 존재하는 느낌"을 만든다.

**레퍼런스 무드**
- 꽃/나무 위에 컬러 사각 블록이 충돌하는 콜라주 스타일 (ref_01.jpg)
- 도트·할프톤·픽셀 처리가 섞인 그리드 분열 (Image 1 — green botanical)
- 글리치+데이터 오버레이 느낌 (Image 2 — pink flower glitch)
- 핀터레스트 레퍼런스: 같은 나무가 계절·처리별로 패널 분열 (mov 영상)

## 렌더링 모드 7가지

| 모드 | 설명 | 핵심 파라미터 |
|---|---|---|
| `original` | 원본 사진 그대로 | — |
| `dot` | 밝기 기반 사각 도트 | `step = RW/100` |
| `halftone` | 원형 할프톤 | `step = RW/80` |
| `ascii` | 문자 아스키아트 (`@#S%?*+;:,.`) | `cw = RW/130` |
| `pixel` | 모자이크 픽셀화 | `sz = RW/55` |
| `glitch` | RGB 채널 분리 + 수평 슬라이스 | 슬라이스 22개 |
| `duotone` | 그린 계열 2색 매핑 | dark:#001400 / light:#b4db50 |

## 패널 레이아웃 구조

```
genPanels() 자동 생성 — 결정론적 seed (Math.sin 기반)

극소형 28개  ← 말 중심(0.50, 0.45) 가우시안 분포 집중
소형   20개  ← 중간 링 (원형 배치, dist 0.22~0.40)
중형   14개  ← 바깥 링 (dist 0.36~0.50)
대형    6개  ← 가장자리 고정 좌표
```

- 패널 **겹침 허용** (z-order: 큰 패널 먼저, 작은 패널 위)
- 모든 패널이 **동일한 전역 패럴랙스** 오프셋 사용 → 말 실루엣 연속성 유지
- 패럴랙스: `ox = (mx-0.5)*18`, `oy = (my-0.5)*14`

## 인터랙션

| 트리거 | 동작 |
|---|---|
| 자동 (3.2초) | 전체 패널 순차 전환 (큰→작은 순, 65ms 딜레이) |
| 마우스 무브 | 전체 이미지 패럴랙스 |
| 클릭 | 카메라 플래시 + 즉시 전환 트리거 |
| 패널 호버 | 해당 패널 테두리 하이라이트 |

**전환 메커니즘**  
`easeInOut` 1.1초 / 전환 중간(t: 0.38~0.62)에 흰 플래시 / 패널별 모드 시퀀스는 `shuffleSeq(idx)`로 개별 셔플

## 뷰파인더 UI 레이어

```
#vf-frame      — 전체 외곽 얇은 테두리 (inset 12px)
#c-tl/tr/bl/br — 코너 L자 마커 (28px, 1.5px 흰선)
#crosshair     — 중앙 십자선 + 원형 dot
#vf-circle     — 중앙 원형 링
#scanline      — 수직 스캔라인 애니 (4s loop)
#rec           — 좌상단 REC 인디케이터 (빨간 점멸)
#expo          — 우하단 카메라 노출 정보 (F/1.8 1/125s ISO400)
```
