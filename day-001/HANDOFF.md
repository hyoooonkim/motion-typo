# Day 001 — 내 맘이란 추는 나를 더 깊게, 더 깊게 붙잡아

## 작품 개요
- **텍스트**: "내 맘이란 추는 나를 더 깊게, 더 깊게 붙잡아"
- **컨셉**: 중력 침강 · 압착 왜곡
- **인터랙션**: 자동 재생

## 핵심 메커니즘
사용자가 일러스트레이터로 만든 텍스트 이미지(`text.png`)를 가로 슬라이스로 잘라서 중력 법칙으로 땅에 파고드는 느낌을 연출. 슬라이스들이 위에서 아래로 압축되며 땅을 뚫고 들어가듯 쌓임.

## 에셋
- `text.png` — 메인 텍스트 이미지 (사용자 제작)
- `thumb.png` — 썸네일
- `무제-1.ai` — 일러스트 원본

## 기술 스택
- 순수 HTML/CSS/JS (외부 라이브러리 없음)
- Canvas API로 이미지 슬라이스 왜곡
- `file://` 로컬 실행 가능

## 디자인 방향
- 배경: 블랙
- 텍스트 색감: 화이트
- 무거운 중력 · 땅을 파고드는 물리 감각
