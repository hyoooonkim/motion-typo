# 프로젝트 인계 문서 — 10대들은 인공지능과 무슨 대화를 할까?

## 🎯 프로젝트 개요

데일리 바이브코딩 Vol.01 인터랙티브 아트웍.
iMessage 말풍선으로 사람 옆모습 실루엣을 만들어 **5명의 AI × 1명의 10대**가 대화하는 모습을 표현.

**기사 출처**: https://maily.so/oz.talking/posts/d5rywql4z1w
**주제**: 10대들이 AI 챗봇(ChatGPT, Gemini, Character.AI 등)과 어떤 대화를 하는지

## 🎨 디자인 컨셉

### 레퍼런스 이미지
원본 밈: 파란 iMessage 말풍선들이 세로로 쌓여서 옆모습 얼굴 실루엣을 만들고, 괄호 `))))` / `((((` 기호가 말풍선 안에 채워져 머리카락을 표현하는 형태.

### 핵심 디자인 규칙
1. **옆모습 프로필**: 말풍선 스택으로 얼굴만 표현 (몸은 없음)
2. **AI ↔ 10대 마주봄**: AI는 왼쪽(오른쪽 봄), 10대는 오른쪽(왼쪽 봄)
3. **이모지는 신체부위만** 사용 (`👁️` `👂` `👄` `💋` `👅`) — 얼굴 표정 이모지(`🤔` `😊` 등) 금지
4. **액세서리는 독립된 말풍선에** 배치 (눈 위에 안경 겹치기 금지)
5. **괄호 기호 = 머리카락**:
   - 상단 3줄: 꽉 찬 머리 (density 1.0)
   - 눈/귀/입 줄: 뒤통수 쪽에만 짧게 (0.25~0.45)
   - 턱 아래: 머리 없음 (0)
6. **AI별 색상 + 글리프로 차별화**:
   | AI | 색상 | 글리프 | 느낌 |
   |---|---|---|---|
   | Claude | `#C96442` 오렌지 | `)` `(` | 사려깊은 |
   | Cursor | `#1a1a1a` 블랙 | `>` `<` | 코드 브라켓 |
   | ChatGPT | `#10A37F` 그린 | `·` `·` | 수다스러운 |
   | Gemini | `#4285F4` 블루 | `✦` `✦` | 반짝이 |
   | Character.AI | `#D946AD` 마젠타 | `*` `*` | 롤플레이 |

### 미구현 / 고려사항 ⚠️
- 말풍선 꼬리가 아직 완벽히 예쁘지 않을 수 있음 (SVG path curve로 구현)
  - 코드 위치: `buildCharacter()` 함수 내 `if (r.role === 'top')` 블록
  - 현재 방식: cubic bezier path 2개 조합
- 대화 타이밍/스텝이 다소 기계적 (자연스럽게 변주 가능)
- 사운드 볼륨이 고정
- 모바일에서 화면 작을 때 레이아웃 압축 개선 여지

## 🏗️ 코드 구조 가이드

### SVG 좌표 시스템
```
VB_W = 480, VB_H = 560
AI 캐릭터: spine(척추) = x:70 (왼쪽 정렬, 오른쪽 봄)
Teen 캐릭터: spine = x:410 (오른쪽 정렬, 왼쪽 봄)
```

### 핵심 함수
- `buildCharacter(el, side, persona)` — SVG 렌더링 (말풍선 + 글리프 + 이모지)
- `animateTalking(el, persona, frame)` — 입 모양 교체 애니메이션
- `transitionToNextAI()` — AI 전환 + 페이드아웃
- `speakTurn(side, duration)` — 한 턴 대화 재생 + 사운드

### 데이터 구조
```js
// ROWS: 11개 말풍선 (head → neck)
const ROWS = [
  { width: 220, role: 'top' },      // 정수리 (꼬리 있음)
  { width: 260, role: 'fore' },     // 이마
  { width: 240, role: 'brow' },     // 눈썹
  { width: 260, role: 'eyes' },     // 👁️ 눈
  { width: 380, role: 'ear' },      // 👂 귀 (튀어나옴)
  { width: 240, role: 'cheek' },    // 볼
  { width: 280, role: 'mouth' },    // 👄 입
  { width: 240, role: 'chin' },     // 턱
  { width: 170, role: 'acc' },      // 액세서리 (넥타이/가면 등)
  { width: 130, role: 'neck' },     // 목
  { width: 150, role: 'bot' }       // 바닥
];

// HAIR_DENSITY: 머리카락 밀도 (0~1)
const HAIR_DENSITY = {
  'top': 1.0, 'fore': 1.0, 'brow': 1.0,

  'chin': 0, 'acc': 0, 'neck': 0, 'bot': 0

```

## 💬 히스토리 (주요 반복 작업)

사용자와 10+ 차례 핑퐁하며 다음 이슈들을 해결:
1. 초기엔 iPhone 목업을 그렸는데 → 제거 요청
2. 몸통+다리까지 그렸는데 → "얼굴만" 요청
3. 사람 옆모습이 제대로 안 나옴 → 레퍼런스 이미지 재참조
4. 이모지 겹침 (안경+눈) → 분리 배치
5. 텍스트 기호가 너무 많음 → **머리카락 컨셉**으로 밀도 조절
6. AI 전환시 이모지 잔존 → 페이드아웃 통합
7. 말풍선 꼬리 디자인 여러 차례 반복 (polygon → circle → path)
8. 배경 그레이 → 화이트
9. 타이틀 폰트 Pretendard 900, 29px로 조정
