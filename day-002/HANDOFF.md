# Day 002 — '텍스트 힙'의 시대, 요즘은 이렇게 읽는다

## 작품 개요
- **텍스트**: "'텍스트 힙'의 시대, 요즘은 이렇게 읽는다"
- **컨셉**: 인터랙티브 책장 타이포그래피 · 도서 대출 카드 위 그래피티 무드
- **인터랙션**: 호버 · 드래그 · 사운드

## 핵심 메커니즘
사용자가 일러스트로 만든 SVG(`book3.svg`)를 인라인 삽입. 책장에 세워진 책들이 각 글자를 이루며, 호버/드래그로 책 순서를 바꿀 수 있음.

- **호버**: 마우스 올리면 책이 덜컥덜컥 흔들리며 효과음
- **드래그**: 책을 잡아서 순서 바꾸기 가능. 드래그 시 주변 책들이 벌어지며 공간 생김
- **중력 법칙**: 책을 옮기면 주변 책들이 자연스럽게 기울거나 쓰러짐
- `origCenters` 배열로 원본 위치 기준 계산

## 에셋
- `book3.svg` — 최종 사용 SVG (사용자 제작, 컬러 조정 완료)
- `book.svg`, `book2.svg` — 이전 버전 (참고용)
- `test.ai`, `test.svg` 등 — 작업 중 산출물
- `화면 기록 2026-04-17 오전 10.24.20.mov` — 데모 영상

## 기술 스택
- 순수 HTML/CSS/JS
- 인라인 SVG (로컬 `file://`에서 fetch 안 되므로)
- Web Audio API로 효과음 합성

## 디자인 방향
- 도서 대출 카드 위에 그래피티 하는 느낌
- 형광 그린 포인트 (텍스트 '텍스트 힙의 시대'도 책으로 취급)
- 책선반 자체는 고정, 책들만 움직임

## 주요 이슈/러닝
- 드래그 시 다른 책이 같이 따라다니는 버그 → `origCenters`로 원본 위치 기준 계산으로 해결
- `file://` 환경에서 외부 SVG fetch 불가 → 인라인 삽입 방식으로 전환
