# Motion Typography — Daily Media Art

매일 하나의 텍스트 구절을 모션 타이포그래피 미디어아트로 구현하는 프로젝트.

## 구조

```
motion-typo/
├── index.html          ← 갤러리 (전체 작품 목록)
├── day-001/index.html  ← 개별 작품
├── day-002/index.html
└── ...
```

## 작품 목록

| Day | 텍스트 | 컨셉 | 인터랙션 |
|-----|--------|------|----------|
| 001 | "모든 단단한 것은 녹아 공기 속으로 사라진다" | 파티클 분해 | 마우스 호버로 분해 가속 |

## 실행 방법

각 폴더의 `index.html`을 브라우저에서 열면 됩니다. 외부 의존성 없이 단일 파일로 동작합니다.
