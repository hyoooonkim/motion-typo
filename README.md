## 인덱스

메인 페이지(`index.html`)는 워크룸프레스 스타일의 에디토리얼 레이아웃:

- **상단 자동 가로 스크롤** — 전 작품 썸네일이 무한 루프. 마우스 호버 시 정지
- **하단 3단 그리드** — 이미지 + 비대칭 텍스트 카드 (day · 태그 · 타이틀 · 컨셉 · 날짜)
- 로고(`motion typography`)가 홈 링크

## 실행 방법

일부 작품(특히 day-009)은 로컬 이미지를 `getImageData`로 읽기 때문에
`file://`로 열면 CORS 차단으로 렌더가 실패합니다. **로컬 서버로 실행**을 권장:

```bash
cd motion-typo
python3 -m http.server 8765
# → http://localhost:8765/
```

각 작품 폴더의 `HANDOFF.md`에 세부 구조·작업 히스토리가 정리되어 있습니다.
